## Setup

1. Fork This Repository

2. Clone Your Repository
  * `git clone https://github.com/<YOUR-USERNAME>/Dad-Jokes.git`

3. While In The Directory Install The Packages By Using
  * `npm install` or `npm i`

4. Run Server By
  * `npm start`
